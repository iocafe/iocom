The selectwifi is small library to allow setting wifi network name (SSID) and password (pre shared key) over blue tooth or serial port.
This library can be used when connecting IO board without display/keyboard to WiFi network. Functionality is limited to WiFi connection
settigs: Since the blue tooth and serial port connections are unsecure, there is no support for software updates or further network 
or security configuration.
